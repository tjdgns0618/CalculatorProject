# Java로 구현한 계산기

--- 
## 계산기 이용법
### 1. 양의 정수를 2개 입력해주세요.
### 2. +, - , *, / 사칙연산 기호 중 하나를 입력해주세요.
### 3. 결과 출력 후 exit를 입력해 주시면 프로그램이 종료됩니다.
### 4. 결과 출력 후 remove를 입력해 주시면 컬렉션에 가장 오래된 값을 지웁니다.

---

변수 이름중에 사칙연산 기호를 입력 받는 operation과 사칙연산 기호 상수 enum operator랑 너무 햇갈리네요....  

### 각 스텝별 구현 사항

---

#### 스텝 1
- [x] 양의 정수 2개 입력받기
- [x] 양의 정수가 아닐 경우 다시 입력 받기
- [x] 정수 타입이 아닐 경우 다시 입력 받기
- [x] 사칙연산 기호 입력 받기
- [x] 사칙연산 기호가 아닐 경우 다시 입력 받기
- [x] 사칙연산 입력 실패시 사칙연산 입력 다시하기
- [x] 사칙연산 기호에 맞는 계산 진행하기
- [x] 나눗셈 연산에서 분모에 0이 입력될 경우 숫자입력으로 돌아가기
- [x] 반복문을 이용하여 계산 반복하기
- [x] 반복 여부 입력 받기

#### 스텝 2

---

- [x] 사칙연산 수행후 결과값을 반환하는 메서드 만들기
- [x] 연산 결과를 저장하는 컬렉션 타입 필드 만들기
- [x] 계산을 다른 클래스를 이용해서 작동시키기
- [x] 연산 결과를 컬렉션에 저장하기
- [x] 연산 결과를 저장한 컬렉션을 캡슐화 하기
- [x] 컬렉션에서 제일 오래된 저장값을 삭제시키는 기능 만들기

#### 스텝 3

---

- [x] Enum을 이용해서 사칙연산 상수집합 만들기
- [x] Enum을 이용해서 사칙연산 수행하는 메서드 만들기
- [ ] 실수를 입력 받을 수 있게 제네릭 클래스로 수정
- [ ] 조회 메서드 구현 (입력 받은 값보다 큰 저장값 출력, 람다&스트림)
